## Ключевые концепции

1. **DOM** - Document Object Model - представление HTML в виде объектов
2. **События** - взаимодействие пользователя с интерфейсом
3. **Случайность** - разнообразие для естественного вида
4. **Flexbox** - гибкое расположение элементов
5. **Адаптивность** - работа на разных устройствах
